Intelligent & Personal Framework for HTTP Adaptive Streaming

Welcome to IPFHAS framework

This project is an implementation of my PhD thesis.

This framework propose to adapt streamed multimedia to each user to be closed to their own expectations.

The goal is to be compatible with the main HTTP Adaptive Streaming systems as HLS (HTTP Live Streaming), DASH (Dynamic Adaptive Streaming over HTTP) and HDS (HTTP Dynamic Streaming).

Make an IPFHAS to your users :)
